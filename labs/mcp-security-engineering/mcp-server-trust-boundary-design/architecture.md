# MCP Server Trust Boundary Design — Architecture Note

## Purpose

This LAB teaches learners how to design and review MCP server trust boundaries.

It focuses on:

- Server identity
- Server ownership
- Environment and approval state
- Allowed resources
- Tool exposure boundaries
- Data classification
- Approval and revocation paths
- Reviewer-safe evidence capture
