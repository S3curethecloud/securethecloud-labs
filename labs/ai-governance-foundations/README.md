# AI Governance Foundations Lab

## Objective

Teach students how to evaluate an AI system before adoption, deployment, or sensitive-data access.

## What You Will Build

A simulated AI governance intake package containing:

- AI system summary
- Business purpose
- Data classification
- Model risk tier
- Human review requirement
- Approval evidence checklist
- Executive decision memo outline

## Student Tasks

1. Define the AI use case.
2. Identify sensitive data exposure.
3. Assign a risk tier.
4. Map review requirements.
5. Produce an approval evidence checklist.
6. Write a short decision memo.

## Safety Boundary

This is a simulation-only lab. It does not authorize real AI deployment, connect to production systems, create users, issue tokens, or enforce runtime policy.
