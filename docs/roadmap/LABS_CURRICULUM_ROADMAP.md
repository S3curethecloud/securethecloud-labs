# SecureTheCloud Labs Curriculum Roadmap

SecureTheCloud Labs is a hands-on educational platform for cloud security, AI governance, zero-trust architecture, evidence engineering, and compliance readiness.

## Learning Tracks

1. Cloud Security Foundations
2. Zero Trust Identity
3. Secure Compute Foundations
4. AI Governance Foundations
5. Evidence Engineering
6. SOC 2 Readiness Labs
7. Agent Security and Runtime Governance

## Platform Rules

- Labs must be educational and reproducible.
- Labs must clearly distinguish simulation from production enforcement.
- Labs must not claim certification, audit completion, or live enforcement unless separately authorized.
- Labs must preserve student-safe execution boundaries.

## Next Build Targets

- Track landing pages
- Lab difficulty metadata
- Estimated completion time
- Evidence checklist per lab
- Student-safe local execution notes
