
Cloud Security Operations L2 Track Module Roadmap

Status: Planned / Not Implemented

Planned Modules
1. Cloud Security Operations Overview

Introduce cloud security operations, detection, triage, evidence, escalation, and response narratives.

2. Cloud Event and Signal Classification

Teach how to classify cloud events by source, severity, identity, workload, asset, tenant, and risk context.

3. IAM Activity Triage

Teach suspicious IAM activity review: role assumption, privilege changes, access key activity, policy changes, failed access, and anomalous identity behavior.

4. Workload and Network Signal Triage

Teach workload and network signal review: compute events, storage access, public exposure, network paths, service behavior, and suspicious workload activity.

5. Cloud Control-Plane Incident Evidence

Teach how to collect and organize evidence for control-plane incidents without mutating runtime systems.

6. Detection Rule Reasoning and False Positive Review

Teach how to reason about detections, false positives, false negatives, tuning, and evidence thresholds.

7. Incident Timeline and Escalation Narrative

Teach how to build a timeline, record decisions, escalate appropriately, and explain impact.

8. Executive Security Summary

Teach how to convert operational evidence into concise executive summaries without overstating certainty or enforcement.

9. Cloud Security Operations Evidence Harness

Teach repeatable evidence packages for detection, triage, escalation, and portfolio-ready response artifacts.
